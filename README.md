# 프로젝트 개요
### 프로젝트 명 : 소설 웹사이트
### 프로젝트 설명: 
작가와 독자 간의 소통과 참여를 불러일으키는 소설 웹 페이지 구현
사용자 편의성을 고려한 직관적이고 간결한 인터페이스 제공합니다.
다양한 장르와 테마에 대한 웹 소설 구독 서비스을 지원하는 다양한 기능을 제공하고 있습니다

### 주요기능

+ 로그인기능
+ 리뷰 게시판 CRUD 구현
+ 이미지 슬라이드
+ 소설 정보와 삽화 등록
+ 소설 작성 기능
+ 댓글 기능 구현
